# ICARUS RS Signatur-Rechner

Deploy-fertige Vite/React-App für Vercel.

## Lokal testen

```bash
npm install
npm run dev
```

## Build

```bash
npm install
npm run build
```

## Vercel

1. Projekt bei Vercel importieren oder hochladen
2. Framework: **Vite**
3. Build Command: `npm run build`
4. Output Directory: `dist`

Fertig.
